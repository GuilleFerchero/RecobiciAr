# RecobiciAr

**RecobiciAr** es un paquete de R diseñado para facilitar el acceso,
descarga y análisis de los datos del sistema de bicicletas públicas de
la Ciudad de Buenos Aires (**ECOBICI**).

El objetivo es proveer una interfaz unificada tanto para la **API en
tiempo real** (GBFS) como para los **datasets históricos** estáticos.
